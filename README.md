# Apex-Tracker

#### [Apex Legends](https://www.ea.com/games/apex-legends) Stats Tracker. API provided by: [Tracker Network](https://tracker.gg/)
Provides players leaderboard to track gameplay history from Origin, PSN & Xbox.
